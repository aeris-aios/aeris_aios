AI MODEL ARCHITECTURE

Use Anthropic as the default provider.

Configuration:
- PRIMARY_MODEL_ALIAS=sonnet
- HIGH_REASONING_MODEL_ALIAS=opus
- FALLBACK_MODEL_ALIAS=haiku

Requirements:
- do not hardcode dated model names
- use alias-based runtime model selection
- use sonnet for standard everyday workflows
- use opus for higher-complexity reasoning tasks
- include safe fallback logic
- log model used, latency, errors, and usage
- create admin AI settings for model routing, usage, and limits

MODEL ROUTING REQUIREMENTS

Implement intelligent routing:
- Sonnet for normal chat, writing, summarizing, research formatting, ad variants, content drafting
- Opus for deeper research synthesis, strategic recommendations, campaign architecture, advanced reasoning, multi-step analysis
- Haiku or fallback for lightweight operations if needed

Allow:
- automatic routing
- manual “Deep Think” mode
- plan-based access controls for advanced reasoning
